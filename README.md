# Odin Recipes

This is my first Odin Project web development project.

## Project Structure

- `index.html` — homepage with links to recipes
- `recipes/` — folder containing individual recipe pages
- `images/` — images for each recipe

## Recipes Included

- Chicken Tikka Masala
- Pizza
- Cheese Burger

## Usage

Open `index.html` in a browser to navigate the recipes.
